THE PROJECT

This project is based around personal learning goals. During the individual project, a start on front-end framework research was done. I thought it would be a good idea to apply this further into a personal project.

My mother owns an acupuncture company, and the website is really outdated. After being asked to revamp the website, I had the idea that I could start learning a front-end framework simultaneously with this.

The website is pure front-end, and acts as an invitation card to customers, which means that there is no back-end attached to the website. This meant that I could fully immerse into the front-end framework, and build something fresh and new.

PROCESS

Approach

The current website is outdated, but the information is still valid. This means that I could use the information architecture that is currently existent in website, and apply this to my revamped design.

Everything else was open for interpretation for me, as long as the new website had a "modern" feel.

This meant that I could fully go out in the design part as well, but I had to keep in consideration that what I designed, is also technically reachable.

Execution

For this project I had set up one main research question:

  • How do create a modernized website for my mothers company with the help of a front-end framework?
