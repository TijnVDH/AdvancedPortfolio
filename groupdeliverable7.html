PROJECT ASSIGNMENT

Design Challenge: How can physical exercise teachers in primary school get more insight into the sound levels and stress during the class?

What problem are we solving: PE/Gym teachers need to be aware of the effects of loud sounds, because they can affect their hearing and stress levels.

Desired situation: If the project is executed well, the result (desired situation) would be that PE/Gym teachers are able to be easily made aware of noise levels in gym halls.

SCOPE

The end deliverables for the group project are as follows:

  • UX Research Documents
  • Low-fidelity until high-fidelity design process.
  • Fully clickable prototype.
  • Proof of concepts

RESEARCH QUESTIONS

The main research question for this project is:

  • How can we use a smartphone app to give insights to young exercise teachers on hearing damage caused by loud noises during PE lessons?

There are also several sub-questions related to this main question:

  • Where does most of the noise pollution come from?
  • In what way can we visualize data in order to make sure it is insightful for our target audience?
  • What type of data will help the user become completely aware of potential hearing damage?
  • Are there specific in-app features that can help teachers to organize PE classes with behavioural changes to decrease noise levels?
  • How can we test the UI/UX of the app?
  • How can we create a unified corporate identity to ensure a better user experience?
  • How can the app's UI/UX be optimised for both Apple and Android users?

For each of these sub-questions, specific research methods are chosen from the CMD methods, which can be seen in the project document itself.
