INDIVIDUAL PROJECT

METHODS AND TOOLING

DELIVERABLE CONTAINING THE METHODS AND TOOLING THAT HAS BEEN USED DURING THE INDIVIDUAL PROJECT.

FIGMA

I used Figma for my entire designing process. This includes the wireframes, until the final prototype which has been tested with users. Using knowledge I gained on auto-layouts, styles and components during my internship, I was able to smoothly work through this process.

NOTION

I used Notion for my entire project management. Notion is a very flexible tool which I started using during my studies at ICT. For individual projects like these it is an ideal program to use. This is also the place where I kept track of my backlog and SCRUM sprints.

GITHUB

I used Github for my version management. It is the tool that is constantly being used to keep up with the latest updates of my portfolio.
