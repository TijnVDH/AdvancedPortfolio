UML USE CASE DIAGRAM

DELIVERABLE CONTAINING A WORKED OUT SCHEMATIC UML CLASS DIAGRAM FOR A PERSONAL WEBSITE.

INTRO

For my personal project, I planned on working on my mothers personal website. For this website, I have created a UML use case diagram. I came up with a question for myself:

  • How do I create an overview of the architecture of the current website, so that I can apply the same logic to the redesigned website?

WHICH DIAGRAM?

I wanted to create a form of schematic overview of the website. Initially I wanted to make a choice between two visualisations:

  • C4 Model
  • UML Class Diagram

Because I already had done an attempt on a C4 Model in portfolio development, I wanted to try something else for this website. I had created UML Class Diagrams before, and I started out with this. However, after adding some classes I realized that an UML Class Diagram for a simple website as this was overly complex.

At this time I was looking at YouTube tutorials about UML class diagrams, but then I stumbled upon this video on UML use case diagrams.

After watching this video, I noticed that such a diagram would visualize what I was working towards neatly.
